# Deis Tester

Docker images containing a buildbot master and slaves, for continuous integration testing of
the [Deis](http://deis.io) open source PaaS.

These Docker images are based on the official [ubuntu:12.04](https://index.docker.io/_/ubuntu/) base image.

Please add any issues you find with this software to the parent [Deis project](https://github.com/opdemand/deis/issues).
